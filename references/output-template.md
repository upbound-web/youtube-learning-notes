# Output Template Reference

## Document Structure

The output markdown follows this structure. **Only include sections that are relevant to the video content and user's TELOS context.**

```markdown
# [Video/Content Title] - [Brief Topic Description]

**Source:** YouTube - [Channel/Speaker Name]
**Author:** [Speaker(s)]
**Date Consumed:** [Today's Date]
**Link:** [YouTube URL]
**Type:** Video/Podcast

---

## Summary
[2-4 sentence overview of the core message and value of the content]

---

## Key Takeaways

**Main ideas:**
1. **[Idea Title]** - [Explanation]
2. **[Idea Title]** - [Explanation]
[3-5 main ideas]

**Important quotes:**
> "[Direct quote from video]"
> "[Another impactful quote]"
[Include 3-6 memorable quotes]

---

## Insights & Thoughts

[DYNAMIC SECTIONS - Only include what's relevant]

**How this applies to [relevant TELOS area]:**
- [Specific application point]
- [Another application]

**Questions this raises:**
- [Thought-provoking question based on content + TELOS context]
- [Another question]

---

## Action Items

**What I'll do differently:**
- [ ] [Specific actionable task tied to TELOS goals/challenges]
- [ ] [Another action]

**Further research needed:**
- [ ] [Topic to explore further]

---

## Connections

**Related to:**
- [Connection to TELOS goals, projects, or challenges]

**Similar concepts:**
- [Related books, ideas, frameworks mentioned or implied]

**Tags for finding later:**
#[tag1] #[tag2] #[tag3]

---

## Personal Rating

**Usefulness (1-10):** [Rating]
**Would recommend:** [Yes/No + brief reason]
**Who would benefit:** [Target audience description]

---

## Additional Context
[Any extra notes, speaker background, key definitions, or supplementary information]
```

## Section Selection Guidelines

### Always Include (with or without TELOS)
- Title header with source info
- Summary
- Key Takeaways (main ideas + quotes)
- Tags
- Personal Rating

### Include Based on Video Content (No TELOS)

When no TELOS file is provided, select sections based purely on video topic:

| If video discusses... | Include section... |
|----------------------|-------------------|
| Business/entrepreneurship | "Business Applications" |
| Technical skills | "Technical Applications" |
| Personal development | "Life Applications" |
| Relationships/social | "Relationship Insights" |
| Health/fitness | "Health & Wellness Applications" |
| Career/professional | "Career Implications" |
| Philosophy/mindset | "Mindset Shifts" |
| Finance/investing | "Financial Applications" |

Generate generic but useful:
- Action items based on video content
- Questions the content raises
- Connections to well-known books/concepts

### Include Based on Relevance (With TELOS)

| If video discusses... | Include section... | Map to TELOS... |
|----------------------|-------------------|-----------------|
| Business/entrepreneurship | "How this applies to my business/work" | Problems, Mission, Projects |
| Technical skills | "Technical applications" | Projects, Challenges |
| Personal development | "How this applies to my life" | Goals, Challenges |
| Relationships/social | "Relationship insights" | Goals (personal), Challenges |
| Health/fitness | "Health & wellness applications" | Goals, Wisdom |
| Learning/education | "Learning applications" | Ideas, Projects |
| Career/professional | "Career implications" | Goals, Challenges, Mission |
| Creative work | "Creative applications" | Projects, Ideas |
| Philosophy/mindset | "Mindset shifts" | Wisdom, Narratives |
| Finance/investing | "Financial applications" | Goals, Predictions |

### Mapping Video Insights to TELOS

When TELOS is provided, generating "Insights & Thoughts" and "Action Items":

1. **Scan TELOS for relevance:**
   - Do any PROBLEMS connect to video themes?
   - Does the MISSION align with content?
   - Can any GOALS be advanced by this knowledge?
   - Do CHALLENGES have solutions in this content?
   - Are there PROJECT applications?

2. **Generate personalized questions:**
   - Based on CHALLENGES: "Am I doing X that the video warns against?"
   - Based on GOALS: "How can this help me achieve G1/G2/etc?"
   - Based on IDEAS: "Does this confirm or challenge my idea about X?"

3. **Create actionable items:**
   - Tie directly to specific GOALS or CHALLENGES
   - Be concrete and time-bound where possible
   - Include "further research" for deep topics

### When No TELOS Provided

Generate useful notes without personalization:

1. **Generic but valuable questions:**
   - "How does this apply to my current situation?"
   - "What's the first step I could take?"
   - "Where am I already doing this well/poorly?"

2. **Broadly applicable action items:**
   - Based on the video's recommendations
   - Concrete next steps anyone could take
   - Further research topics

3. **Skip:**
   - TELOS ID references (G1, C2, P1, etc.)
   - "How this applies to my business" (use "Business Applications" instead)
   - Overly personalized language

## Quality Standards

- **Quotes:** Must be actual quotes from the video, not paraphrased
- **Actions:** Must be specific and actionable, not vague
- **Connections:** Reference actual TELOS items when relevant (G1, C2, P1, etc.)
- **Rating:** Honest assessment based on value delivered
- **Length:** Comprehensive but not padded - every line should add value
